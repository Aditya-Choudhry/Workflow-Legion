# Demo Script

Workflow Legion is a Band-native cyber incident command room.

The key point: Band is not a notification layer. Band is where the agents collaborate.

## Flow

1. Start incident simulation.
2. Alert Triage Agent posts the alert in Band.
3. Triage Agent mentions Threat Intel and Forensics.
4. Threat Intel enriches indicators.
5. Forensics builds a timeline.
6. Compliance reviews audit risk.
7. Incident Commander posts final decision.
8. Dashboard displays the incident report.
